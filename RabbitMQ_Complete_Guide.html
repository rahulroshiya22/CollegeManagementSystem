📚 What is RabbitMQ?

RabbitMQ is a message broker - think of it as a post office for your code! One service sends a message, and RabbitMQ delivers it to the right service.

🚀 Asynchronous

Services don't wait for each other

🔗 Loose Coupling

Services work independently

📦 Reliable

Messages safely stored until delivered

⚙️ Quick Setup - CloudAMQP

Step 1: Install Package

dotnet add package RabbitMQ.Client --version 7.2.0

Step 2: Add Connection String

appsettings.json
{
  "CloudAMQP": {
    "ConnectionString": "amqps://user:pass@server/vhost"
  }
}

Step 3: Publish Messages

await _messageBus.PublishAsync("exchange-name", message);
